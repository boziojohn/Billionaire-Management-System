# Billionaire-Management-System
This project was developed as part of the Programming II course at university. It is a C-based application that manages billionaire data using a circular doubly linked list. The program supports insertion, deletion, searching with filters, and sorting based on wealth, birthdate, and time. It also includes file handling for saving/loading data and generates unique bank account numbers. The project demonstrates key concepts such as dynamic memory allocation, pointers, string processing, and data structures
